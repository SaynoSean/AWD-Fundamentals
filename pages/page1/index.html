The 4 Fundamentals

Variable Declaration: Defines a variable and its data type to store values during program execution.

Input and Output: Enables interaction between the program and the user by receiving inputs and displaying outputs.

Conditional Statement: Allows decision-making in the program by executing code based on whether a condition is true or false.

Looping Statement: Repeats a block of code multiple times based on a condition or a set number of iterations.
